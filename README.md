# The Hum - LD46
## Description
Feed the bonfire.  
_Keep it alive._  

![screenshot](./screenshots/ingame.png)

## Controls
| Key          | Action                  |
|:------------:|:----------------------- |
| `A` / `D`    | Move left / right       |
| `W` / `S`    | Climb ladders up / down |
| `SPACE`      | Jump                    |
| `E`          | Interact                |
| `D`, `ENTER` | Start game              |

## Development
Using [`amethyst`] and [`deathframe`].  
Setup with [`deathframe-template`].
